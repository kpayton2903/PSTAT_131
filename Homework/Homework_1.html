Machine Learning Main Ideas ¶

1).¶

Define supervised and unsupervised learning. What are the difference(s) between them?

Supervised learning is a method of machine learning in which for each observation from our dataset, there is an associated response variable we are attempting to predict. In contrast, unsupervised learning is a machine learning method that once again uses observations, but has no associated response variable we are attempting to predict. Unsupervised learning is typically used to understand the relationships between our observations.

2).¶

Explain the difference between a regression model and a classification model, specifically in the context of machine learning.

Both regression and classification models are types of supervised machine learning methods in which we use observations to predict some response variable. In regression our response variable is a quantitative variable, or a numerical variable. Whereas in classification, our response variable is a qualitative variable, or a categorical variable. Both of these methods are attempting to do the same thing but with different types of responses.

3).¶

Name two commonly used metrics for regression ML problems. Name two commonly used metrics for classification ML problems.

Regression:

MSE = $\frac{1}{n}\sum^n_{i=1}(y_i-\hat{f}(x_i))^2$

The mean squared error measures how close our predicted responses are to the true value for that specific observation of our training data.

$E(y_0-\hat{f}(x_0))^2 = Var(\hat{f}(x_0)) + [Bias(\hat{f}(x_0))]^2 + Var(\epsilon)$

The expected test mean squared error measures the average test mean squared error we would find if we were to use many different large training sets to calculate different values of f.

Classification:

Error rate = $\frac{1}{n}\sum^n_{i=1}I(y_i \neq \hat{y}_i)$

The training error rate measures the proportion of observations that are given the wrong predictor class label.

Bayes error rate = $1-E[max_j Pr(Y=j|X))$

The bayes error rate gives us an error measurement based on the Bayes Classifier, which assigns the class for which $Pr(Y=j|X=x_0)$ is largest. This error rate is essentially the same as the irreducible error.

All the above equations and similar descriptions come from our textbook.

4).¶

As discussed, statistical models can be used for different purposes. These purposes can generally be classified into the following three categories. Provide a brief description of each.

  • Descriptive models:

  • Inferential models:

  • Predictive models:

Descriptive Models:

These models are used to summarize or describe patterns found in data. They focus on understanding features of a dataset such as its distribution, averages, or relationships between variables. These models don't attempt to make predictions about future datapoints.

Inferential Models:

These models aim to make generalizations about a population based on a sample of data. They are able to help estimate parameters and test hypotheses to determine if the observed data can be generalized to a broader context.

Predictive Models:

These models focus primarily on predicting future outcomes or values based on historical data. They are able to use patterns learned from training data to predict future trends.
